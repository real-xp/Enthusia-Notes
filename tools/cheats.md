# Cheats
There exist a lot of cheats that have been made for Enthusia. A lot of these are to make life easier, such as a custom B spec driver AI or even a Debug car.

These cheats can be found in [this github repository](https://github.com/real-xp/enthusia-professional-racing-cheats)

The most notable ones are 
- Debug Car 
  - Basically a hidden unused function that was never meant to me used by normal user but was left in the game
  - Controls
    - DPAD = Step
    - Right Stick = Move
    - Left Stick = Rotate
    - Hold L2 and move = Move faster
- Auto Driver (AI)
  - This is basically B Spec of this game. This allows cars to be driven by the AI itself
  - There are two type of AI
    - AIRS - Faster AI, More Risk Taker AI
    - AI - Slower AI, Less Risk Taker AI
- Instant Win
  - Basically makes you win instantly
- Unlock Cars / Tracks
  - Basically unlocks cars and tracks for Free Racing, also unlocks tracks, same with DR
- S Rank DR
  - Basically S Ranks all DR stages
