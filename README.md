# Enthusia Notes And Discoveries

> If you believe you can, therefore you can.
> 
> ~ Anonymous

This is a small repository that contains a lot of notes and discoveries related to Enthusia Professional Racing (PS2 Video Game made by Konami)

A lot of this was found on the internet like japanese archives or by digging through game code or by doing other stuff.

It takes a long time for me to update this repo usually.

Please note that posting anything that includes decompiled code is not allowed.
